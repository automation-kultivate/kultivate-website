Accessibility

Accessibility Statement

Last updated: 17 June 2026

Our commitment

KULTIVATE Music GmbH aims to make this website accessible and usable for as many people as possible. We want visitors to be able to access information about our company, music projects, broadcast formats and playlist ecosystem with minimal barriers.

Current status

This website is currently an informational company and label website. It does not currently provide an online shop, direct checkout, booking system or consumer contract conclusion through the website.

The website has been designed with semantic HTML structure, keyboard-accessible navigation, visible focus states, readable contrast, responsive layouts and reduced-motion support where possible.

Accessibility measures

  • Semantic page structure with header, navigation, main and footer landmarks.
  • Keyboard-accessible links and navigation.
  • Responsive layout for desktop and mobile screen sizes.
  • Alternative text for meaningful images.
  • Visible focus styles for interactive elements.
  • Reduced-motion support for users who prefer less animation.

Known limitations

The website has not yet been formally audited by an external accessibility specialist. Some visual effects, animations and image-heavy sections may require further refinement as the site evolves. Placeholder images and third-party media integrations may be replaced or adjusted over time.

Future services

If the website later offers direct bookings, paid services, online forms, embedded media players, checkout functionality or other interactive services, this accessibility statement and the technical implementation will be reviewed and updated accordingly.

Feedback
